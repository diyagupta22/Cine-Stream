# prompts i used:-

## Prompt 1 – Infinite Scroll

How to implement infinite scrolling in a React movie app using Intersection Observer so that more movies load automatically when the user scrolls down.

## Prompt 2 – Debounce Search

How to create a debounce hook in React so that the search input waits 500ms before filtering movies.

## Prompt 3 – Favorites with LocalStorage

How to store selected movies in localStorage so they remain saved even after refreshing the page.

## Prompt 4 – Remove from Favorites

How to remove a movie from the favorites list by filtering the movie array using its id.

## Prompt 5 – Lazy Loading Images

How to load movie poster images only when they appear on the screen using `loading="lazy"`.

## Prompt 6 – Mood Based Recommendation

How to implement a simple mood-based movie recommendation feature where users type a mood and the app suggests a related movie.

## Prompt 7 – UI Improvements

How to improve the UI of movie cards using grid layout, hover effects, and better styling to make the interface look more modern.
